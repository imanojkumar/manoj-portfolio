# Manoj Kumar - Personal Website & Portfolio

Welcome to the source code for my personal website and digital portfolio. This site serves as a central hub for my work, research, and insights across **Data Science, Artificial Intelligence, Digital Transformation, and ESG**.

## 🚀 Tech Stack

This website is generated statically using [Quarto](https://quarto.org/), featuring a highly customized, responsive UI built from scratch.

-   **Framework:** Quarto
-   **Styling:** Custom Modular CSS (CSS Variables, Flexbox, CSS Grid)
-   **Interactivity:** Vanilla JavaScript (IntersectionObserver for scroll animations)
-   **Hosting:** GitHub Pages

## 📁 Project Architecture

-   `_quarto.yml`: Main Quarto configuration and navigation routing.
-   `*.qmd`: Quarto markdown files for individual pages (Home, Books, Talks, etc.).
-   `styles/`: Modular CSS system and JavaScript logic.
-   `partials/`: Reusable HTML blocks (e.g., the custom interactive footer).
-   `docs/`: The compiled, production-ready website served by GitHub Pages.

## 🛠️ Local Development

To run this project locally, ensure you have Quarto installed, navigate to the root directory, and run:

``` bash
quarto preview
```
